Working

Technologies used: Node.Js, HTML, CSS, Bootstrap, JavaScript, JQuery.
This website works by taking live data from a website, cricapi.com, where we are provided with the data, which is well formatted and styled.
A Unique Key is provided, which is inserted in a url, on request of which, data is fetched, formatted, and displayed.
A Request module of Node.Js is used to request the url, data is fetched, filtered, mapped, and then send to the html page. Using Express module of Node.Js, server is created. Other Node.Js modules include path and body-parser.
